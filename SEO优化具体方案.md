# SEO 优化具体方案

## 核心问题诊断

根据 Bing SEO 工具报告，您的网站存在以下主要问题：

1. **标签/分类页面标题过短** ← 最急迫
2. **部分页面 Meta 描述不足或缺失** ← 次急迫  
3. **缺乏高质量入站链接** ← 长期问题
4. **某些自动生成页面无搜索结果**

---

## 🎯 优化方案

对 `docs/.vuepress/theme.mjs` 进行以下改进：

### 方案 2：为自动生成页面添加 Meta 描述

VuePress Theme Hope 可能不会自动为这些页面生成 Meta 描述。如果上面的 `blogLocales` 修改后仍没有效果，可以考虑创建自定义组件或插件。

**临时解决方案：在自动生成的页面中注入 Meta **

在 `docs/.vuepress/config.mjs` 中添加（需要检查 VuePress 2 是否支持此方式）：

```javascript
head: [
  // ... 既有的 head 配置
  
  // 为自动生成页面提前定义 Meta 描述的默认值
  ["meta", {
    name: "description",
    content: "科学上网与翻墙机场推荐知识库，涵盖机场选择、流媒体解锁、AI工具使用等完整指南"
  }],
],
```

但这只会应用于首页。对于标签/分类页面，需要：

**更好的方案**：等待或升级到支持更好 Meta 管理的 VuePress 版本

**建议的标签清理**：

|------|----------|------|

| 科学上网 | 多 | ✅ 保留 |
| AI工具 | 多 | ✅ 保留 |
| 流媒体 | 多 | ✅ 保留 |
| [单独标签如 "OpenClaw"] | 1 | ⚠️ 考虑合并到 "AI工具" |

### 方案 5：改进 robots.txt 和 sitemap

#### 检查 `docs/.vuepress/public/robots.txt`

```txt
User-agent: *
Allow: /

# 允许 Bing 爬虫
User-agent: Bingbot
Allow: /

# 禁止爬取一些文件
Disallow: /.vuepress/
Disallow: /node_modules/

# 指向 sitemap
Sitemap: https://clash-jichang.com/sitemap.xml
```

### 方案 6：增加入站链接（长期策略）

#### 短期（1-2周）
1. ✅ **联系友链网站**
   - 经常逛的相关博客、论坛
   - 在 `/links` 页面中已列出部分，主动联系他们交换友链
   - 目标：增加 5-10 个质量友链

2. **参与社区讨论**
   - 在知乎相关问题中回答（附上链接）
   - 在 V2EX 分享最新的翻墙、机场、AI 相关话题
   - 在 GitHub 项目讨论中推荐您的指南

3. **撰写最新热点文章**
   - 紧贴最新行业动态（如您已有的 "2026年4月中转拔线潮" 文章）
   - 易获得转发和包含链接的引用

#### 长期（持续）
1. **寻求媒体提及**
   - 联系科技博客、播客、YouTube 频道
   - 提供独家数据或深度分析

2. **建立权威性内容**
   - 创建综合性指南（已在做）
   - 发布原创测评数据
   - 进行工具对比实验
