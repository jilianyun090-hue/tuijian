## 如何选择合租平台

1. **看运营历史** — 运营超过 2 年且有大量交易记录的平台可信度更高。新平台即使价格诱人也应保持警惕。
2. **看退款政策** — 正规平台都有明确的退款/补偿机制。在购买前务必阅读平台的服务条款。
3. **看售后渠道** — 优先选择有 Telegram 售后群或 7x24 工单系统的平台。
4. **看支付方式** — 支持支付宝直连比通过第三方代付更安全。

## 为什么不推荐在闲鱼购买

| 问题 | 具体表现 |
| :--- | :--- |
| 跑路风险 | 个人卖家随时可能停止续费，账号失效后无法追责 |
| 黑卡风险 | 部分低价账号使用盗刷信用卡注册，被追回后可能牵连您的设备 |
| 无售后 | 出现问题只能通过平台仲裁，效率低且结果不可控 |
| 隐私泄露 | 个人卖家可以随时登录您的账号，查看观看记录 |
