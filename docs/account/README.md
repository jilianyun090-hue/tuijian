## 各平台合租详细说明

### Netflix 合租

Netflix 高级版（4K）支持最多 4 个独立 Profile，每个 Profile 有独立的观看记录、推荐算法和下载列表。通过合租平台购买的独立 Profile 与自己购买使用完全一致，不会互相干扰。

**注意事项**：
- 不要修改主账号密码或邮箱，否则会影响所有成员
- 如需换区观看（如追日剧），切换节点即可，Profile 内容不受影响
- Netflix 对合租较为友好，只要不频繁更换 IP 地区即可正常使用

### ChatGPT Plus 合租

ChatGPT Plus 合租通常为独享账号，即你拥有一个属于自己的 OpenAI 账号，由合租平台完成 Plus 升级付款。

**注意事项**：
- 账号通常由合租平台协助注册，使用前确认是否可修改密码
- 避免同时在多个设备大量使用，可能触发账号保护机制
- 推荐通过银河录像局购买，封号有赔偿保证

### YouTube Premium 合租

YouTube Premium 家庭组合租是最具性价比的方式，每人只需 ¥5-10/月 即可享受全站无广告、后台播放和离线下载。

**使用体验**：无广告效果立竿见影，重度 YouTube 用户强烈推荐。详见 [YouTube Premium 订阅指南](../streaming/youtube-guide.md)。

### Spotify 合租

Spotify 家庭组最多容纳 6 人，每人有独立账号，听歌记录和播放列表完全独立。合租价格约 ¥8-12/月。

详见 [Spotify Premium 订阅指南](../streaming/spotify-guide.md)。

## 合租避坑指南

::: danger 高风险行为，请避免
- ❌ 通过闲鱼、拼多多等不正规渠道购买合租账号
- ❌ 更改他人账号的主密码、手机/邮箱绑定
- ❌ 在合租账号中存放重要个人信息
- ❌ 同时让家人/朋友使用同一个 Profile（可能触发多设备限制）
:::

::: tip 推荐做法
- ✅ 通过认证合租平台购买，有售后和封号赔偿
- ✅ 使用独立 Profile/账号，不与他人共享同一 Profile
- ✅ 按月购买，满意后再换年付，避免长期锁定
- ✅ 保留购买截图和订单记录，便于售后维权
:::
