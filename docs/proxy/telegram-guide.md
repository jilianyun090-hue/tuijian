## Telegram 是什

Telegram 又名电报，也被称飞机"，是一款注重隐私保护和高效沟通的即时通讯应用。由俄罗斯企业家 Pavel Durov 2013 年创立，现由 Telegram FZ-LLC 公司运营，总部位于阿联酋迪拜。

Telegram **隐私保护****跨平台同****高可用**著称。支持消息云端同步、端到端加密聊天、超大群组和频道、强大的机器人生态，以及大文件传输能力，非常适合个人和团队协作。

此外，Telegram 对内容的限制较少，能有效避免违禁导致的沟通不畅，这也是许多用户选择它的重要原因。

## Telegram 常见问题 (FAQ)

**Q1：没有手机号能用 Telegram 吗？**

Telegram 注册与登录必须使用手机号接收验证码。若不想使用私人号码，可使用 [接码平台](/ai/sms-guide.html) 获取虚拟号码进行注册。

**Q2：Telegram WhatsApp 有什么区别？**

Telegram 在隐私可控、自定义程度、机器人生态与文件大小上限方面更灵活；频道/群规模也更大。

**Q3：Telegram 消息能否跨设备同步？**

可以。手机、电脑可同时在线，消息在云端同步。但"秘密聊天"除外，其内容仅保存在本地设备。

**Q4：为什么我的新号容易受限？**

为防止垃圾信息，系统会对新账号的加群、私聊等行为进行限制。建议：
- 完善个人资料
- 减缓操作频率
- 通过邀请链接加入群。
- 如被限制，可联系官方机器`@SpamBot` 了解原因与解除时。

## Telegram 安全使用指南

- ***开启两步验证：** 为你的账号设置独立密码，这是最重要的安全防。
- ***管理设备列表** 定期检查在线设备，及时清理陌生或不再使用的会话
- ***善用秘密聊天** 讨论敏感内容时，优先使用端到端加密的秘密聊天功能，并设置消息自动销。
- ***遵守社区规则** 避免骚扰式加群或群发广告，遵守平台规则与当地法律政策
