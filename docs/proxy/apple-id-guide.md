## 🛠️ 核心注册步骤

### 第一步：开启科学上网环境
确保你的节点已切换至**美国**。推荐使用 [2026年优质机场推荐](/airport/) 中的高速节点，确保注册过程不掉线。

### 第二步：更改设备地区
1.  打开 iPhone **“设置”** > **“通用”** > **“语言与地区”**。
2.  将“地区”改为 **“美国”**。

### 第三步：通过 App Store 启动注册
**关键点：** 不要直接在网页注册，请从 App Store 入口进入。

1.  打开 App Store，点击右上角头像，退出当前账号。
2.  搜索一个**免费应用**（例如 TikTok），点击“获取”。
3.  在弹出的登录框中选择 **“创建新 Apple ID”**。

![注册入口示意图](/assets/img/guide/apple_id_registration_step_1_1773148359829.png)

### 第四步：填写账号信息
1.  输入准备好的邮箱和设置密码。
2.  付款方式选择 **“None” (无)**。
    > **💡 注意**：如果你没看到 "None" 选项，通常是因为你没有从下载免费应用的路径进入，或者代理节点不够纯净。

| 字段 | 示例值 (俄勒冈州) |

| City | Portland |

| Zip Code | 97205 |
| Phone | 503-xxx-xxxx |

![地址填写示例](/assets/img/guide/apple_id_us_address_example_1773148378755.png)

### 第五步：验证账户
1.  分别输入邮箱收到的验证码和手机收到的验证码。
2.  验证成功后，你的美区 Apple ID 就正式开通了！
