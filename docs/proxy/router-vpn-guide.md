## 一、什么是路由器翻墙？

路由器翻墙是指在**无线路由器硬件端直接配置 VPN 或代理**，让所有连上 Wi-Fi 的设备无需安装任何软件即可「科学上网」。

与传统的在每台设备上安装翻墙客户端不同，路由器翻墙是一种**网关级别的解决方案**，从网络入口直接实现全局代理。

---

## 二、路由器 VPN 翻墙的基本要求

### 1. 硬件兼容性

路由器需满足以下任一条件：
- ✅ 原生支持 VPN 客户端（如 ASUS 梅林系列）
- ✅ 支持刷入 **OpenWRT**、**Merlin（梅林）** 等第三方固件

**推荐硬件规格：**

| 配置项 | 最低要求 | 推荐配置 |
| :--- | :--- | :--- |
| CPU | ≥ 1.0 GHz | ≥ 1.4 GHz（双核） |
| 内存 | ≥ 128 MB | ≥ 256 MB |
| 闪存 | ≥ 16 MB | ≥ 128 MB |

### 2. 机场兼容性

你的机场需要支持以下主流路由器协议：
- **Clash** / **Mihomo** — 最常用
- **Xray / V2Ray** — 通用方案
- **WireGuard** — 速度快，延迟低

并提供可导出的**订阅链接**，方便路由器自动更新节点和规则。

| 优势 | 说明 |

## 四、路由器翻墙的主流方案

### 方案一：OpenWRT + OpenClash / Passwall

适用于大多数支持刷机的路由器，生态丰富，插件众多。

**推荐插件：**
- **OpenClash** — 图形化界面，配置简单
- **Passwall** — 功能强大，支持多种协议
- **ShellCrash** — 轻量级，适合低配路由器

### 方案二：梅林固件 + Clash 插件

适用于 ASUS（华硕）系列路由器，稳定性极佳。

### 方案三：旁路由方案

使用一台低功耗迷你主机（如 R2S、R4S）作为旁路由，不改变主路由器设置。

## 六、主流品牌配置思路

### 🔹 华为路由器

多数华为家用款**无内置 VPN 客户端**。建议方案：
- 刷入 OpenWRT（进阶用户）
- 作为**旁路由**接入（推荐）

### 🔹 小米路由器

**AX3600 / AX6000** 性能强悍，是性价比之选：
1. 通过 SSH 开启 root 权限
2. 刷入 OpenWRT 固件
3. 安装 Passwall 或 OpenClash 插件

### 🔹 TP-Link 路由器

仅 **Archer 系列**部分型号原生支持 OpenVPN 客户端：
- 直接在设置中上传机场提供的 VPN 配置文件即可
- 其他型号需刷 OpenWRT

## 七、常见问题 FAQ

**Q：路由器翻墙会影响网速吗？**

会有一定影响，取决于路由器的 CPU 性能和加密协议的复杂度。推荐选择硬件性能较强的路由器。

**Q：所有设备都必须走代理吗？**

不是。大多数方案支持**分流模式**，可以指定哪些设备或网站走代理，其余直连。

**Q：路由器翻墙安全吗？**

只要使用正规机场和最新固件，安全性是有保障的。建议定期更新固件和插件。
