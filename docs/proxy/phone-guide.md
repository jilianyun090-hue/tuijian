手机科学上网的核心分为两部分：**客户端（软件）** 和 **订阅（节点）**。

1.  **客户端（APP）**：安装在手机上的流量转换工具。
2.  **订阅（节点）**：从机场服务商处购买的连接凭证。没有节点？请参考 [便宜好用的翻墙机场推荐](/airport/)。

## 一、 Android (安卓) 手机翻墙指南

安卓系统开放度高，配置非常简单。

### 1. 推荐客户端
我们首选推荐 **Clash Meta for Android**，其稳定性与分流能力最为出色。
👉 [Android 客户端下载与详细教程](../airport/client-android.md)

### 2. 快速上手
1.  **导入订阅**：打开 APP，进入“配置”界面，选择“从 URL 导入”，粘贴你的订阅链接。
2.  **启动代理**：回到首页，点击“启动”按钮。手机会弹出 VPN 连接请求，点击“确定/允许”即可。

## 二、 iOS (iPhone / iPad) 手机翻墙指南

苹果手机由于 App Store 规则限制，需要额外的美区 ID 才能下载工具。

### 1. 必备工具
iOS 平台的黄金标准工具是 **Shadowrocket (小火箭)**。

### 2. 操作前提：美区 Apple ID
中国区 App Store 已下架所有此类软件。你需要登录港区或美区 Apple ID 进行下载。
- 👉 [2026 最新美区 Apple ID 获取与合租指南](/proxy/apple-id-guide.html)

### 3. 连接步骤
1.  **下载软件**：在美区商店搜索并付费下载 Shadowrocket。
2.  **一键导入**：在机场后台点击“一键导入小火箭”，软件会自动打开并填充配置。
3.  **开启开关**：点击首页最顶部的开关，首次连接允许创建 VPN 配置即可畅享。

## 三、 手机翻墙常见问题 (FAQ)

**Q：为什么连上代理后，微信和抖音加载变慢了？**
- **原因**：你可能开启了“全局模式”，导致访问国内流量也绕道了国外。
- **对策**：请务必勾选 **“规则模式” (Rule/Config Mode)**。该模式会智能判断，国内流量直接访问，国外流量才走代理。

**Q：只有连着 WiFi 能翻，切换到流量就不行了？**
- 请检查手机权限，是否禁用了该代理 APP 的“使用移动数据”。
- 检查 APN 设置是否允许 VPN 流量。

**Q：手机耗电量会增加吗？**
- 由于代理工具需要后台运行并加密流量，耗电量会轻微增加，这属于正常现象。
