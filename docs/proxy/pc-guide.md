在开始实操之前，你只需要理解两个核心概念，就能轻松掌握科学上网：

1.  **客户端（软件）**：这是一辆“车”。你需要下载安装在电脑上，比如目前最推荐的 **Clash Verge Rev**。
2.  **订阅/节点（机场）**：这是“油”。你需要从服务商处购买订阅链接，导入软件后，网络才能通畅。

## 一、 Windows 电脑翻墙指南

Windows 是目前翻墙最成熟的平台。我们强烈建议使用开源、高性能且界面现代的 **Clash Verge Rev**。

### 1. 下载与安装
千万不要下载已停止更新的老旧版本（如原 Clash for Windows）。
👉 [Windows 客户端下载与详细配置教程](../airport/client-windows.md)

### 2. 核心操作步骤
1.  **导入订阅**：打开软件，点击左侧的 **订阅 (Subscription)**，粘贴你购买的机场链接，点击 **导入 (Import)**。
2.  **开启代理**：在左侧 **代理 (Proxies)** 选项卡中，将模式设置为 **Rule (规则模式)**。
3.  **接管流量**：进入 **设置 (Settings)**，打开 **System Proxy (系统代理)** 开关。看到开关变绿，即可畅享国际互联网。

## 二、 Mac (macOS) 电脑翻墙指南

Mac 平台的配置逻辑与 Windows 基本一致，但由于系统权限更严格，需要注意授权。

### 1. 软件安装
Mac 用户同样最推荐使用 **Clash Verge Rev**（完美支持 Intel 和 M 系列芯片）。
👉 [macOS 客户端下载与详细配置教程](../airport/client-windows.md)

### 2. 操作要点
1.  **初次启动**：若提示“无法验证开发者”，请前往 `系统设置 -> 隐私与安全性` 中点击 `仍要打开`。
2.  **配置与启动**：导入订阅后，在顶部状态栏的软件图标中勾选 **设置为系统代理 (Set as System Proxy)**。

## 三、 常见问题排查 (FAQ)

**Q：软件变绿了，但还是上不了网？**
- 请检查软件内的“内核”是否启动。
- 在代理界面进行“节点测速”，如果节点列表全是 `Timeout`，说明你的订阅已过期或线路已被屏蔽。

**Q：关闭软件后，电脑变砖（无法访问任何国内网页）？**
- 这是因为软件在异常退出时没能及时关闭“系统代理”。
- **解决方法**：重新打开 Clash Verge Rev，先手动关闭“系统代理”开关，然后再退出软件。

**Q：手机和电脑可以共用一个机场账号吗？**
- **完全可以**。大部分优质机场都支持多设备同时在线。你只需要把同一个链接分别导入手机和电脑即可。
