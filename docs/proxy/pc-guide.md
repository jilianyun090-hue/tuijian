# 电脑如何翻墙？2026年Windows与Mac科学上网新手指南

在开始之前，你只需要理解两个核心概念，就能明白翻墙的原理：

1.  **客户端（软件）**：这是一辆“车”。你需要下载并安装在电脑上的软件，比如 **Clash Verge Rev**。软件本身通常是开源免费的。
2.  **订阅/节点（机场）**：这是“油”。你需要购买的服务，你会得到一个订阅链接。把它导入到软件里，你的“车”才能跑起来。

## 一、 Windows 电脑如何翻墙

Windows 是目前使用最广泛的系统，翻墙方式非常成熟。目前最推荐使用的是开源、免费且功能强大的 **Clash Verge Rev**。

### 1. 下载与安装
不要下载已停更的老旧版本。推荐使用新版的 Clash Verge Rev。
- [Windows 客户端下载与配置教程](../airport/client-windows.md)

### 2. 操作步骤
1.  **导入订阅**：打开 Clash Verge Rev，点击左侧的 **订阅 (Subscription)**，粘贴从机场获取的链接并点击 **导入 (Import)**。
2.  **开启代理**：点击左侧 **代理 (Proxies)**，选择 **Rule (规则模式)**。
3.  **系统接管**：点击 **设置 (Settings)**，打开 **System Proxy (系统代理)** 开关。看到开关变绿，即表示翻墙成功。

## 二、 Mac (macOS) 电脑如何翻墙

苹果电脑配置过程与 Windows 类似，但由于 macOS 安全机制，安装软件时可能需要额外授权。

### 1. 下载客户端
Mac 用户同样推荐使用 **Clash Verge Rev**（支持 Intel 和 M 芯片）。
- [macOS 客户端下载与配置教程](../airport/client-windows.md)

### 2. 操作步骤
1.  **安装软件**：下载 .dmg 文件，将图标拖入应用程序文件夹。如果提示“无法验证开发者”，请到 `系统设置 -> 隐私与安全性` 中点击 `仍要打开`。
2.  **配置节点**：打开软件，找到 **订阅** 栏目，粘贴链接并下载配置。
3.  **启动连接**：在状态栏找到软件图标，勾选 **设置为系统代理 (Set as System Proxy)**。

## 三、 常见问题 (FAQ)

### Q1: 软件打开了，为什么还是上不了网？
- 确保软件内的“系统代理”开关已开启。
- 在代理界面点击“测速”，如果节点全是超时，说明订阅已过期。

### Q2: 关闭软件后，电脑无法上网了怎么办？
- Clash 必须先关闭“系统代理”开关，再退出软件。
- 如果已经退出了，请进入 `Windows 设置 -> 网络和 Internet -> 代理`，手动关闭“使用代理服务器”。

### Q3: 手机和电脑可以共用一个订阅吗？
- 可以。大部分机场支持多设备同时在线。你只需要把同一个订阅链接分别导入手机和电脑即可。
