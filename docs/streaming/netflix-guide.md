# Netflix 观看全指南

Netflix (奈飞/网飞) 是全球流媒体巨头，提供海量片库、顶级原创内容（如《纸牌屋》、《怪奇物语》）以及 4K HDR 极致视听体验。

## 如何观看 4K 高清

观看 Netflix 4K 需要满足硬件、软件及网络三方面的要求：

| 设备类型 | 推荐软件/方式 | 最高画质 |
| :--- | :--- | :--- |
| **iOS / Android** | 官方 APP (需海外账号下载) |  tùy 设备支持 |
| **Mac** | Safari 浏览器 | 4K HDR |
| **Windows 11** | Edge 浏览器 / Netflix UWP 应用 | 4K HDR |
| **电视盒子** | Apple TV 4K / Nvidia Shield Pro | 4K HDR |

### 流量消耗参考
*   **480p**: 约 0.8 GB/小时
*   **1080p**: 约 2-2.5 GB/小时
*   **4K**: 约 3.5-7 GB/小时

## 常见问题排查

### 1. 代理/解除代理限制 (Proxy Error)
Netflix 会检测你的 IP 是否为数据中心 IP。如果使用普通机场节点，可能会出现“您似乎正在使用解除封锁程序或代理”的提示。
**解决方法**：选择支持“Netflix 解锁”或“原生 IP”的机场节点。

### 2. 土耳其区账号限制
土耳其区价格最低，但目前新号往往有 IP 限制（S7111-1331）。建议使用土耳其节点观看一段时间，或选择其他区域（如埃及、巴基斯坦）合租。
