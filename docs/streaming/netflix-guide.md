# Netflix 观看全指南

## 什么是 Netflix（奈飞）？新手快速了解

Netflix（奈飞），全球流媒体娱乐的巨头，彻底颠覆了人们的观影方式。从 DVD 租赁起家，到如今坐拥超过 2.7 亿付费用户，覆盖 190 多个国家和地区，Netflix 的发展史堪称传奇。

**Netflix = 海量片库 + 顶级原创 + 极致体验**

- **海量片库**：超过 1.3 万部影视作品，好莱坞大片、获奖佳作、热门美剧、精品纪录片……应有尽有
- **顶级原创**：《纸牌屋》《怪奇物语》《王冠》《鱿鱼游戏》……现象级原创剧集层出不穷
- **4K HDR 超高清画质**：在家也能享受影院级视听盛宴
- **多语言字幕/配音**：轻松跨越语言障碍，大部分内容均有简体中文字幕
- **智能推荐算法**：精准推荐你喜欢的影片，告别剧荒
- **多设备无缝观看**：手机、平板、电脑、电视，随时随地，想看就看

尽管 Netflix 尚未正式进入中国大陆，但通过科学上网即可畅享其优质内容。

## 国内如何看 Netflix

想要在国内流畅使用 Netflix 并获得最佳观影体验，你需要做好以下准备：

1. **获取 Netflix 账号**：拥有一个账号是观影的前提。推荐通过 [合租平台](../account/README.md) 购买独立 Profile，性价比最高
2. **选择支持 Netflix 解锁的机场**：借助机场提供的原生 IP 专属线路，突破网络限制，稳定连接 Netflix。→ [查看机场推荐](/airport/)
3. **选择合适的设备和浏览器**：不同设备能观看的最高画质不同，详见下文

## Netflix 如何看 4K 高清

### iOS 设备（iPhone / iPad）

- **要求**：下载海外 App Store 中的 Netflix 官方 App（需海外 Apple ID）
- **画质**：iPhone 12 及以上机型支持 4K HDR + Dolby Vision
- **注意**：不要使用国区 Apple ID 下载，国区无此 App

### Mac 设备：Safari 浏览器解锁 4K HDR

- **要求**：在 Mac 上使用 **Safari 浏览器**访问 netflix.com
- **画质**：搭载 Apple Silicon（M1/M2/M3）的 Mac 可达 4K HDR + Dolby Vision
- **注意**：Chrome、Firefox 等浏览器最高仅支持 720p，Edge 在 Mac 上最高 1080p

### Android 设备

- **要求**：下载 Netflix 官方 APK（需从官网或 Play Store 获取，非国内应用商店）
- **画质**：取决于设备的 **Widevine** DRM 认证等级：
  - **L1（最高级）**：可播放 4K 或 1080p，大多数旗舰手机支持
  - **L3（最低级）**：最高仅支持 SD 标清，常见于部分国内品牌手机

### Windows 11 PC：Edge 浏览器 + UWP 应用

- **Edge 浏览器**：访问 netflix.com，最高支持 **1080p**
- **Netflix UWP 桌面应用**（微软商店下载）：支持最高 **4K** HDR，是 PC 看 4K 的最佳方式
- Chrome / Firefox 最高仅 720p

### 安卓电视盒子 / Apple TV

| 设备 | 最高画质 |
| :--- | :--- |
| Apple TV 4K | 4K HDR + Dolby Vision + Dolby Atmos |
| Nvidia Shield Pro | 4K HDR |
| 安卓盒子（支持 Widevine L1）| 4K（视设备而定）|
| Chromecast with Google TV | 4K HDR |

### 智能电视

内置 Netflix 应用的 4K 智能电视（Sony、Samsung、LG 等品牌）通常直接支持 4K HDR 播放，是最无脑的大屏 4K 观影方案。

## Netflix FAQ：常见问题

### 如何查看当前播放的分辨率和码率？

播放视频时，在视频画面双击，然后按下 **Ctrl+Alt+Shift+D**（PC）或通过 Netflix 隐藏菜单可以查看当前分辨率、比特率、CDN 节点等技术信息。

### Netflix 分辨率、码率与流量消耗估算

| 画质 | 分辨率 | 约码率 | 每小时流量 |
| :--- | :--- | :--- | :--- |
| 低清 | 480p | ~0.5 Mbps | ~0.3 GB |
| 标清 | 720p | ~3 Mbps | ~1.3 GB |
| 高清 | 1080p | ~5-8 Mbps | ~2-3 GB |
| 超高清 | 4K HDR | ~15-25 Mbps | ~7 GB |

### 代理能上 Google、YouTube，为什么不能看 Netflix？

Netflix 对 IP 要求非常严格，普通机场的数据中心 IP 会被 Netflix 识别并拦截，出现 M7111-5059 报错。**必须选择标注"Netflix 原生IP解锁"的专属节点**，这类节点使用的是 Netflix 合法授权的住宅 IP 或原生 IP。

→ [推荐支持 Netflix 解锁的机场](/airport/)

### 为什么我不能看 4K 分辨率的 Netflix 视频？

需同时满足：①订阅高级版套餐；②使用支持 4K 的设备和浏览器（见上文）；③网络带宽 ≥ 25 Mbps；④机场节点带宽稳定。

### 如何将 Netflix 界面切换为简体中文

登录后进入账户设置 → 「语言」→ 选择「简体中文」→ 保存。播放内容时，也可在视频播放器的对话框图标中切换字幕语言。

### Netflix 影视剧如何显示简体中文字幕？

点击播放器右下角的「对话框」图标，在字幕选项中选择「中文（简体）」。注意：并非所有内容都有简体中文字幕，热门美剧和原创剧集通常提供中文字幕。

## Netflix 常见错误码及解决方案

| 错误码 | 含义 | 解决方案 |
| :--- | :--- | :--- |
| **M7111-5059** | 检测到代理/解锁工具 | 更换支持 Netflix 原生 IP 的机场节点 |
| **S7111-1331** | 账户无法在此地点使用 | 切换到与账号注册区域一致的节点 |
| **22004** | 此影片在您所在区域不可用 | 切换不同地区节点（如美区、港区） |
| **40102** | 无法在选定设备播放 | 检查 Widevine 认证级别，换设备 |
| **10016-2300** | 下载设备数量过多 | 在账户设置中删除旧下载设备 |
