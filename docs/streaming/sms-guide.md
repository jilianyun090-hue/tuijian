# 接码平台 (SMS Verification) 评测

注册 ChatGPT、Claude、Telegram 等海外服务时，通常需要国外手机号进行短信验证。接码平台提供来自全100 多个国家的临时手机号，用于接收一次性验证码。

## 主流平台对比

| 平台 | 支持支付方式 | 号码纯净 | 适用场景 | 最低充 |
| :--- | :--- | :--- | :--- | :--- |
| [Hero SMS](https://herosms.com/) | **支付/ 微信** |  | Claude / OpenAI 注册 | ~$2 |
| [SMS-Activate](https://sms-activate.org/cn/) | 支付/ 加密货币 |  | Telegram / Microsoft | ~$1 |
| [Onlinesim](https://onlinesim.io/cn/) | 信用/ 加密货币 |  | 租用(按小按天) | ~$3 |
| [5sim](https://5sim.net/) | 支付/ 信用 |  | 批量注册 | ~$1 |

## 各平台详细评

### Hero SMS

面向中文用户优化的接码平台，界面支持中文，支持支付宝和微信充值。号码以实体 SIM 卡为主，通过率较高。推荐用Claude ChatGPT 等风控严格的平台注册。

### SMS-Activate

俄罗斯老牌平台，号池全球最大，覆盖 180+ 个国家。价格极具竞争力，但部分号码为虚拟号，在 Anthropic 等高风控平台可能被拒。适合 Telegram、Microsoft、Google 等大众服务的注册。

### Onlinesim

特色在于"租用"模式——可以租用一个号码几小时甚至几天，期间所有短信都会转发给您。适合需要多次验证的场景 (如注册后还需二次确认)。

## 接码实战流程

以注ChatGPT 为例。

### 步骤一：选择接码平台并充

Hero SMS SMS-Activate 注册账号，充$2-5 (支付。

### 步骤二：选择业务和国

- 搜索 "OpenAI" "ChatGPT" 业务。
- 选择国家。推荐印度尼西亚 (便宜) 或美(通过率高)。
- **关键**：确保接码平台选择的国家与您当前代理节点的国家一致，可显著提高通过率。

### 步骤三：获取号码并填

平台分配一个临时号码。将此号码填ChatGPT 注册页面的手机验证环节。

### 步骤四：等待验证

通常 30 秒内收到短信。将验证码填入注册页面即可。

**如果 2 分钟未收**：点取消"重新获取新号码。大多数平台只在成功接码后才扣费。

## 虚拟vs 实体

| 类型 | 成本 | 通过 | 适用场景 |
| :--- | :--- | :--- | :--- |
| 虚拟(VoIP) | ($0.2-0.5) | (60-80%) | Google、Microsoft、Telegram |
| **实体(Real SIM)** | 较高 ($1-3) | **(90%+)** | **Claude、OpenAI、Twitter/X** |

风控严格的平(尤其Anthropic Claude) 强烈建议使用实体号。

## 长期方案

如果您经常需要海外手机号验证，可以考虑购买一张海外实SIM 卡邮寄至国内长期使用。推荐：

- **英国 giffgaff** 免费申请，每半年发一条短信即可保号。支持全球漫游收短信。
- **美国 Ultra Mobile** $3/月保号套餐，可收美国短信。
