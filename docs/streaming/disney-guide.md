# Disney+ 观看指南

Disney+ 涵盖漫威 (MCU)、星球大战 (Star Wars)、迪士尼经典、皮克斯 (Pixar) 及国家地理等丰富内容。

## 观影优势

*   **合租高性价比**：通过合租平台（如银河录像局），价格约为直订的 1/5。
*   **海量 IMAX Enhanced**：支持大量漫威电影的 IMAX 比例画面。

## 地区选择建议

| 地区 | 特色 | 内容评级 |
| :--- | :--- | :--- |
| **新加坡/香港** | 内含 **Star** 专区，且包含 R21/18+ 成人内容 | 最高级别 |
| **美国** | 内容更新最快，但部分内容需额外购买 (Premier Access) | 普通级别 |

### 如何解锁 18+ 内容？
1.  登录账户，进入“编辑用户档案 (Edit Profile)”。
2.  点击“内容分级 (Content Rating)”，输入密码确认。
3.  选择最高级别（通常为 18+ 或 R21）。

## 硬件要求
需要支持 HDR 的 4K 电视/盒子，且网络带宽至少达到 25Mbps 才能流畅开启 4K 画质。
