## 配置订阅

### 第一步：获取订阅链接

登录您的机场官网，在用户面板找到"订阅链接"Clash 订阅"，复制整URL。如果还没有机场，可以参[优质线路推荐](./README.md) 选购。

### 第二步：导入订阅

1. 打开 Clash Meta for Android 应用。
2. 点击底部菜单***「配置** 选项。
3. 点击右上角的 **** 号按钮。
4. 选择 **「URL** 选项。
5. 将复制的订阅链接粘贴到输入框中，点击 **「确定**。

### 第三步：设置自动更新

1. 自动更新间隔可根据需要设置（推荐 1440 分钟 / 24 小时）。
2. 设置完成后点***「确定**。

### 第四步：保存并选中配置

1. 点击右上角的 **「保存** 按钮。
2. 保存完成后，在配置列表中 **选中** 刚才导入的配置（点击使其高亮）。

## 开始使

1. 返回应用主界面。
2. 点击中间***开关按** 启动代理。
3. 首次启动会弹VPN 连接请求，点***「确定** 授权。
4. 状态栏出现钥匙图标，表示代理已启动。

## 选择节点

1. 点击底部菜单***「代理** 选项。
2. 可以看到所有可用节点列表。
3. 点击某个节点即可切换（推荐选择延迟最低的香港或日本节点）。
4. 长按节点可进行连通性测试。

## 代理模式说明

| 模式 | 说明 | 推荐场景 |
| :--- | :--- | :--- |
| **规则模式** (推荐) | 国内流量直连，国外流量走代理 | 日常使用，最省流 |
| 全局模式 | 所有流量都走代 | 需要全部流量过墙时使用 |
| 直连模式 | 所有流量都直连 | 临时关闭代理 |

建议日常使用保持 **规则模式**，这样访问国内网站（微信、淘宝、B 站等）不受影响，同时访问外网也能正常代理。

## 常见问题 (FAQ)

### Q: 安装 APK 时提不允许安装未知应?

进入手机 **设置 安全 安装未知应用**，找到浏览器或文件管理器，勾允许安装"即可。

### Q: 导入订阅后没有节

1. 检查订阅链接是否完整复制。
2. 检查手机网络是否正常（可以尝试切换 WiFi / 移动数据）。
3. 点击配置旁边的刷新按钮重新拉取。
4. 确认订阅是否已过期。

### Q: 节点全部超时 (Timeout)?

1. 检查系统时间是否准确（代理协议对时间精度敏感）。
2. 确认没有同时开启其VPN 或代理应用。
3. 尝试切换不同的节点测试。

### Q: 耗电量大怎么

这是正常现象。由于安卓系统的 VPN 机制，所有网络流量都经过 Clash 转发，系统会将流量对应的耗电统计Clash 上。实际上 Clash 本身耗电极低。
