## 🛠️ 基本配置逻辑说明 (三步走)

无论您使用哪款软件，配置流程几乎都是一样的：

<br/>

1.  **获取订阅链接**：在您的机场后台找到“便捷导入”或“Clash 订阅链接”并复制。
2.  **导入客户端**：在软件的“配置”或“订阅”模块粘贴链接并下载。
3.  **开启代理**：选择一个响应速度快的节点（如香港、日本），打开“系统代理”开关即可。
