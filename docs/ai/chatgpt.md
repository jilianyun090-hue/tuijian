## 🛡️ 官网实战方案 (进阶用户)

对于希望直接使用 OpenAI 官方网页的用户，请注意以下几点：

1.  **邮箱选择**：必须使用 Gmail 或 Outlook。国内 163、QQ 邮箱可能会被系统拦截。
2.  **网络环境**：节点建议选择美国、日本、新加坡。**严禁使用香港、俄罗斯等封禁地区的节点**。

## 🧠 高效提问 (Prompt) 公式

要想让 AI 给出高质量答案，必须掌握基本的 Prompt 工程：

> **角色 + 任务 + 约束条件 + 背景 = 高质量答案**

**示例：**
*   **普通提问**：帮我写一段代码。
*   **高质量提问**：你现在是一名资深 Python 工程师（**角色**），请帮我写一段处理 CSV 文件的脚本（**任务**），要求代码简洁、包含注释（**约束**），这是为了分析某电商平台的销量数据（**背景**）。
