## 🧠 高效提问 (Prompt) 公式

要想让 AI 给出高质量答案，必须掌握基本的 Prompt 工程：

> **角色 + 任务 + 约束条件 + 背景 = 高质量答案**

**示例：**
*   **普通提问**：帮我写一段代码。
*   **高质量提问**：你现在是一名资深 Python 工程师（**角色**），请帮我写一段处理 CSV 文件的脚本（**任务**），要求代码简洁、包含注释（**约束**），这是为了分析某电商平台的销量数据（**背景**）。

### 常用 Prompt 模板

| 场景 | 模板 |
| :--- | :--- |
| 写作助手 | "你是一名专业[领域]写作者，帮我写一篇关于[主题]的文章，要求[风格/字数/受众]" |
| 代码生成 | "用[语言]实现[功能]，要求[约束条件]，并添加注释" |
| 翻译优化 | "请将以下文本翻译成[目标语言]，保持原文语气，使用地道表达：[文本]" |
| 学习辅助 | "请用通俗易懂的语言解释[概念]，并举3个实际例子" |
